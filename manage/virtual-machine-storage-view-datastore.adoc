= 虚拟机存储视图
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
存储视图显示虚拟机创建的虚拟卷的列表。


NOTE: 此视图适用于至少挂载了一个适用于VMware vSphere管理的Vvol数据存储库相关磁盘的ONTAP工具的虚拟机。

.步骤
. 从vSphere Client导航到虚拟机。
. 在右窗格中选择*Monitor*选项卡。
. 选择* NetApp ONTAP tools*>*存储*。右窗格中将显示*存储*详细信息。您可以查看虚拟机上存在的Vvol列表。
+
您可以使用"管理列"选项隐藏或显示不同的列。
